# Find significant GWAS traits variants defined by p-value <= 5e-8.
# Release: V8
- gwas.file.list: the list of GWAS trait files.
- gwas.filter.sh: the bash script for filtering a given GWAS trait file
- filterBySignificance.5e-8.sh: the batch command for filtering all GWAS trait files. 
- imputed*.filtered.txt: filtered GWAS trait files
- GTEx_V8_GWAS_Share.txt: the mapping of GWAS trait file name to trait term.
